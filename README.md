Парсер статей с Pedsovet.org
Программа для автоматического сбора информации об образовательных статьях с главной страницы портала Pedsovet.org.

Что делает программа
Загружает главную страницу pedsovet.org

Находит карточки со статьями в различных разделах

Извлекает названия статей и ссылки на полные версии

Автоматически преобразует относительные ссылки в абсолютные

Сохраняет результаты в JSON-файл и выводит в консоль

Результаты работы
После запуска создается файл education_articles.json со списком статей. Каждая статья содержит:

Название (article_title)

Ссылку на полную версию (article_url)

Программа также выводит все найденные статьи в консоль с нумерацией.

Требования
Python 3.6+

Библиотеки: beautifulsoup4, requests, lxml
