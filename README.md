# Парсер статей Pedsovet.org

## Описание
Программа для автоматического сбора информации об образовательных статьях с главной страницы портала Pedsovet.org.

## Установка зависимостей

pip install -r requirements.txt

## Запуск программы
bash
python parse_pedagogy.py

## Результаты работы
Создается файл education_articles.json

Данные выводятся в консоль

## Формат данных: название статьи и ссылка на полную версию

## Технические требования
Python 3.6+

## Библиотеки: beautifulsoup4, requests, lxml
